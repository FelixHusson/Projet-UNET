# Projet-UNET
Projet d'introduction à l'imagerie biomédicale – Master MMA Cours dirigé par Irène Kaltenmark

Ce dépôt contient une présentation et une implémentation simplifiée de l’architecture U-Net, un modèle de segmentation d'images largement utilisé en imagerie médicale. Le projet a été réalisé dans le cadre du cours "Introduction à l'imagerie biomédicale" du Master Mathématiques et Applications (MMA).
